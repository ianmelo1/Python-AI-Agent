# Python AI Agent

CodeHelper is an AI assistant for developers, built with Python, LangChain, and Google's Gemini Pro. It provides a suite of tools to streamline your development workflow, from generating sample data to managing project files and analyzing code.

## Features

- **Data Tools**: 
  - `write_json`: Write Python dictionaries to JSON files.
  - `read_json`: Read and parse JSON files.
  - `generate_sample_users`: Create sample user data.
  - `generate_random_data`: Generate random emails, names, companies, and phone numbers.
- **File Tools**:
  - `list_json_files`: List all JSON files in a directory.
  - `verify_file_exists`: Check if a file exists and get its details.
  - `delete_file`: Delete files.
- **Developer Tools**:
  - `create_gitignore`: Generate `.gitignore` files from templates (Python, Node, React, etc.).
  - `create_readme`: Create a `README.md` file.
  - `create_requirements_txt`: Generate a `requirements.txt` file.
  - `read_python_file`: Read Python files with line numbers.
  - `find_todos_in_code`: Find TODO, FIXME, and HACK comments in your code.
  - `count_lines_of_code`: Count the lines of code in your Python files.
  - `search_in_files`: Search for a term in files.
